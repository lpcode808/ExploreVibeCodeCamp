CSS Variables (Design Tokens)

CSS Variables (also called Custom Properties) are the foundation of a maintainable design system. They let you define values once and reuse them everywhere.

What are CSS Variables?

Think of CSS variables like containers that hold values. Instead of writing #e07a5f every time you want coral color, you write var(--accent-coral). If you later decide to change the coral color, you only update it in one place!

Live Example - Color Palette
--accent-coral
--accent-teal
--accent-green
--accent-purple
--link
/* Define variables in :root (applies to whole page) */
:root {
    /* Colors */
    --bg-primary: #2b2b2b;
    --accent-coral: #e07a5f;
    --accent-teal: #5cb8b2;

    /* Typography */
    --font-body: 'Sora', sans-serif;
    --font-mono: 'JetBrains Mono', monospace;

    /* Spacing */
    --spacing-sm: 0.5rem;
    --spacing-md: 1rem;
    --spacing-lg: 2rem;
}

/* Use variables with var() */
.my-element {
    background: var(--bg-primary);
    color: var(--accent-coral);
    font-family: var(--font-body);
    padding: var(--spacing-md);
}

Callout Boxes

Callouts highlight important information. The colored left border draws attention and indicates the type of content.

Live Example - Callouts

Default (Coral): Use for warnings, important notes, or calls to action.

Teal: Use for tips, best practices, or helpful suggestions.

Modal Dialogs

Modals are overlay windows that appear on top of the main content. They're used for focused tasks like search, settings, or viewing details.

Key Modal Concepts

  • Backdrop: A semi-transparent overlay that dims the background
  • Content: The actual modal window with your content
  • Z-index: CSS property that controls stacking order (higher = on top)
  • Visibility toggling: Using classes to show/hide the modal
Modal Structure Diagram
+------------------------------------------+
|  VIEWPORT (your browser window)          |
|  z-index: auto                           |
|  +--------------------------------------+|
|  |  PAGE CONTENT                        ||
|  |  z-index: 1                          ||
|  |  (your main page content)            ||
|  +--------------------------------------+|
|  +--------------------------------------+|
|  |  MODAL CONTAINER (fixed position)    ||
|  |  z-index: 2000                       ||
|  |  +----------------------------------+||
|  |  | BACKDROP (absolute, inset: 0)    |||
|  |  | background: rgba(0,0,0,0.75)     |||
|  |  | backdrop-filter: blur(8px)       |||
|  |  +----------------------------------+||
|  |  +----------------------------------+||
|  |  | MODAL CONTENT (relative)         |||
|  |  | z-index: higher than backdrop    |||
|  |  | Your actual content here         |||
|  |  +----------------------------------+||
|  +--------------------------------------+|
+------------------------------------------+
            
<!-- MODAL HTML STRUCTURE -->
<div id="my-modal" class="modal-hidden">
    <!-- Backdrop - click to close -->
    <div class="modal-backdrop"></div>

    <!-- Content - the actual modal -->
    <div class="modal-content">
        <div class="modal-header">
            <h3 class="modal-title">Modal Title</h3>
            <button class="modal-close" aria-label="Close">
                <!-- X icon -->
            </button>
        </div>
        <div class="modal-body">
            <!-- Your content here -->
        </div>
    </div>
</div>

/* MODAL CSS */
#my-modal {
    position: fixed;         /* Stays in place when scrolling */
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    z-index: 2000;           /* Above everything else */
    display: flex;
    align-items: center;     /* Vertical centering */
    justify-content: center; /* Horizontal centering */
    opacity: 1;
    transition: opacity 0.2s ease;
}

/* Hidden state - toggled by JavaScript */
#my-modal.modal-hidden {
    display: none;
    /* Or use opacity + pointer-events for animation */
}

.modal-backdrop {
    position: absolute;
    inset: 0;                /* Shorthand for top/right/bottom/left: 0 */
    background: rgba(0, 0, 0, 0.75);
    backdrop-filter: blur(8px);  /* Blur the background */
}

.modal-content {
    position: relative;      /* Creates stacking context */
    background: var(--bg-secondary);
    border-radius: 16px;
    width: 90%;
    max-width: 600px;
    max-height: 80vh;
    overflow: hidden;
    box-shadow: 0 20px 60px rgba(0, 0, 0, 0.5);
    border: 1px solid var(--border);

    /* Entrance animation */
    transform: scale(0.95);
    transition: transform 0.2s cubic-bezier(0.16, 1, 0.3, 1);
}

/* Animate content when modal becomes visible */
#my-modal.visible .modal-content {
    transform: scale(1);
}

Reading Progress Bar

A thin bar at the top that shows how far the user has scrolled through the document.

<!-- PROGRESS BAR HTML -->
<div id="reading-progress"></div>

/* PROGRESS BAR CSS */
#reading-progress {
    position: fixed;
    top: 0;
    left: 0;
    height: 3px;
    background: linear-gradient(90deg, var(--accent-coral), var(--accent-teal));
    width: 0%;          /* Updated by JavaScript */
    z-index: 9999;
    transition: width 0.1s;
}

/* JAVASCRIPT TO UPDATE PROGRESS */
window.addEventListener('scroll', () => {
    const docHeight = document.documentElement.scrollHeight - window.innerHeight;
    const scrolled = (window.scrollY / docHeight) * 100;
    document.getElementById('reading-progress').style.width = scrolled + '%';
});

Layout Systems

Modern CSS layouts use Flexbox and Grid. Here's when to use each:

  • Flexbox: One-dimensional layouts (row OR column). Great for navbars, card headers, centering.
  • Grid: Two-dimensional layouts (rows AND columns). Great for page layouts, card grids, galleries.

Card Grid Layout

/* AUTO-FILLING GRID */
.card-grid {
    display: grid;
    /* Auto-fill columns: minimum 320px, max 1fr (equal share) */
    grid-template-columns: repeat(auto-fill, minmax(320px, 1fr));
    gap: 16px;
    align-items: start;  /* Cards align to top */
}

/* This creates a responsive grid that:
   - Has as many columns as fit
   - Each column is at least 320px
   - Columns expand equally to fill space
   - No media queries needed! */

Animations & Transitions

Smooth animations make your UI feel polished. Here are the key patterns:

Transition Basics

/* TRANSITION SYNTAX */
.element {
    /* transition: property duration timing-function; */
    transition: all 0.2s ease;

    /* Or be specific (better performance): */
    transition:
        background-color 0.2s ease,
        transform 0.2s ease,
        box-shadow 0.2s ease;
}

/* TIMING FUNCTIONS */
ease        /* Start slow, speed up, slow down at end (default) */
linear      /* Constant speed */
ease-in     /* Start slow */
ease-out    /* End slow */
ease-in-out /* Start and end slow */
cubic-bezier(0.16, 1, 0.3, 1)  /* Custom curve (bouncy) */

Common Animation Patterns

/* HOVER LIFT EFFECT */
.card {
    transition: transform 0.2s, box-shadow 0.2s;
}

.card:hover {
    transform: translateY(-2px);
    box-shadow: 0 8px 24px rgba(0, 0, 0, 0.3);
}

/* FADE IN/OUT */
.modal {
    opacity: 0;
    pointer-events: none;      /* Can't click when hidden */
    transition: opacity 0.2s ease;
}

.modal.visible {
    opacity: 1;
    pointer-events: auto;
}

/* SCALE ENTRANCE */
.modal-content {
    transform: scale(0.95);
    transition: transform 0.2s cubic-bezier(0.16, 1, 0.3, 1);
}

.modal.visible .modal-content {
    transform: scale(1);
}

/* SLIDE IN FROM LEFT */
.sidebar {
    transform: translateX(-100%);
    transition: transform 0.2s ease;
}

.sidebar.visible {
    transform: translateX(0);
}

JavaScript Patterns

JavaScript brings interactivity to your patterns. Here are the essential functions:

Toggle Classes (Show/Hide)

// Toggle sidebar visibility
document.getElementById('toggle-btn').addEventListener('click', () => {
    document.body.classList.toggle('toc-hidden');
});

// Open/close modal
function openModal(modalId) {
    document.getElementById(modalId).classList.add('visible');
    document.getElementById(modalId).classList.remove('modal-hidden');
}

function closeModal(modalId) {
    document.getElementById(modalId).classList.remove('visible');
    // Or use modal-hidden for immediate hide
    document.getElementById(modalId).classList.add('modal-hidden');
}

// Close modal when clicking backdrop
document.querySelector('.modal-backdrop').addEventListener('click', () => {
    closeModal('my-modal');
});

Scroll Tracking (Active Section)

// Update active TOC item based on scroll position
function updateActiveSection() {
    const sections = document.querySelectorAll('h2[id], .concept-card[id]');
    const tocLinks = document.querySelectorAll('.toc-section a');

    let currentSection = '';

    sections.forEach(section => {
        const rect = section.getBoundingClientRect();
        // If section is near top of viewport
        if (rect.top <= 100) {
            currentSection = section.id;
        }
    });

    // Update TOC highlighting
    tocLinks.forEach(link => {
        link.classList.remove('active');
        if (link.getAttribute('data-id') === currentSection) {
            link.classList.add('active');
        }
    });
}

// Listen for scroll events
window.addEventListener('scroll', updateActiveSection);

Search/Filter

// Simple search filtering
const searchInput = document.getElementById('search-input');
const items = document.querySelectorAll('.searchable-item');

searchInput.addEventListener('input', (e) => {
    const query = e.target.value.toLowerCase();

    items.forEach(item => {
        const text = item.textContent.toLowerCase();
        // Show/hide based on match
        item.style.display = text.includes(query) ? 'block' : 'none';
    });
});

// Highlight matching text
function highlightMatch(text, query) {
    if (!query) return text;
    const regex = new RegExp(`(${query})`, 'gi');
    return text.replace(regex, '<mark>$1</mark>');
}

Keyboard Navigation

// Listen for keyboard shortcuts
document.addEventListener('keydown', (e) => {
    // Cmd/Ctrl + K for search
    if ((e.metaKey || e.ctrlKey) && e.key === 'k') {
        e.preventDefault();  // Prevent browser default
        openModal('search-modal');
    }

    // Escape to close modals
    if (e.key === 'Escape') {
        closeModal('search-modal');
    }

    // T to toggle sidebar
    if (e.key === 't' || e.key === 'T') {
        // Don't trigger if typing in an input
        if (e.target.tagName !== 'INPUT') {
            document.body.classList.toggle('toc-hidden');
        }
    }
});
